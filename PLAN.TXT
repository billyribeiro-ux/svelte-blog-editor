WINDSURF PROMPT — Ultimate SvelteKit Blog Editor (TipTap 3)
PROJECT CONTEXT
You are building a production-grade, headless rich text blog editor for a SvelteKit application. This editor replaces WordPress Classic Editor with something far superior. It uses TipTap 3.x (ProseMirror) as the editor engine, Svelte 5 runes for all reactivity, SvelteKit 2.x for the framework, and TypeScript strict mode throughout.

This is a PE7 / ICT Level 7 grade implementation. No shortcuts, no placeholders, no "you can add more here" comments. Every file must be complete, production-ready, and fully typed.

TECH STACK — NON-NEGOTIABLE
Framework: SvelteKit 2.x with Svelte 5 runes ($state, $derived, $effect, $props, $bindable)
Language: TypeScript strict mode — zero warnings, zero errors, zero any types
Package Manager: pnpm (never npm, never yarn)
Editor Engine: TipTap 3.x (@tiptap/core, @tiptap/pm) — used DIRECTLY, no wrappers
CSS: Scoped CSS with PE7 standards — OKLCH color space, @layer cascade, CSS nesting, logical properties (margin-block-start, padding-inline-end, etc.), design tokens via CSS custom properties. NEVER use Tailwind, DaisyUI, or any CSS framework.
Icons: Iconify with Phosphor or Carbon icon sets only. NEVER use Lucide icons.
Snippets over Slots: Use Svelte 5 {#snippet} / {@render} pattern, never <slot>
No class components: Use runes mode exclusively. No export let, no $:, no stores for new code.

ALL images in the editor MUST have alt text. The MediaModal should enforce this — the Insert button stays disabled until alt text is provided.

TipTap Editor integration pattern for Svelte 5:

<script lang="ts">
  import { onMount, onDestroy } from 'svelte';
  import { Editor } from '@tiptap/core';

  let element = $state<HTMLDivElement>();
  let editorState = $state<{ editor: Editor | null }>({ editor: null });

  onMount(() => {
    editorState.editor = new Editor({
      element: element,
      extensions: [/* ... */],
      content: '...',
      onTransaction: ({ editor }) => {
        editorState = { editor };
      },
    });
  });

  onDestroy(() => {
    editorState.editor?.destroy();
  });
</script>

<div bind:this={element}></div>
